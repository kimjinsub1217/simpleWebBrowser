# 📌simpleWebBrowser
>심플 웹브라우저

## ✔기능
<ul>
<li>웹사이트 접속가능하다.</li>
<li>앞으로가기,뒤로가기 가능하다. </li>
<li>홈 버튼을 눌러 처음으로 돌아갈 수 있다. </li>
<li>웹사이트 로딩을 시각적으로 확인 가능하다. </li>
</ul>
